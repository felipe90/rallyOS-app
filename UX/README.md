# RallyOS UX Documentation

Este folder centraliza la definición de flujos, interacciones y el inventario de componentes de la aplicación.

## Estructura de Documentación

- **[Flows Inventory](flows-inventory.md)**: Listado maestro de macro-flujos y su mapeo a componentes.
- **[UI Patterns](ui-patterns.md)**: Definición de átomos y moléculas visuales (Glassmorphism, High-Tech Clubhouse).
- **[Specific Flows/](flows/)**: Detalle paso a paso de cada flujo crítico.
- **[Desktop & TV Flows/](flows-desktop/)**: Especificaciones para administración en PC y dashboards de TV.

## Metodología: Top-Down
Diseñamos desde la **Intención del Usuario (Vibe)** -> **Flujo de Pantallas** -> **Componentes Atómicos**.
